발급: github.com → Settings → Developer settings → Personal access tokens

처음 설정 가이드
순서대로 따라하세요
1
저장소 생성
github.com/new → 저장소 이름 입력 → Public → Create
그 다음 이 파일들을 업로드: index.html scripts/send_alert.py .github/workflows/meal-alert.yml data/settings.json
2
GitHub Secrets 등록
저장소 → Settings → Secrets and variables → Actions → New repository secret
NEIS_API_KEY · KAKAO_ACCESS_TOKEN · KAKAO_REFRESH_TOKEN · KAKAO_REST_KEY
3
GitHub Pages 활성화
저장소 → Settings → Pages → Source: Deploy from branch → main → Save
약 1분 후 https://사용자명.github.io/저장소명 에서 접속 가능
4
이 페이지에서 설정 완료
위에서 학교·디데이·카카오·시간 설정 후 GitHub에 저장하기 클릭 → 완료! 🎉
